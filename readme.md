## Projeto desenvolvido na "Seção 4: CSS 3" do curso "Desenvolvimento Web Completo 2022".

# Conteúdos desenvolvidos:

Aula 01:
  - Seletores;
  - Aplicação de classes e ids;

  - Formatação de links.

  Aula 02:
  - Aplicação do rodapé;
  - Uso de caixas, com elementos flutuantes;
  - Utilização do "clear".
